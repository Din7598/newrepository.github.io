TCP/IP Tools

What is TCP and what is IP?

TCP stands for Transmission Control Protocol.

IP stands for Internet Protocol.

Protocol protocol protocol... simply put this are rule that define how things should work, from a network perspective how a hardware or software should work logically.

The TCP/IP also known as the internet protocol suite is a set of communication protocols used in the internet network. The TCP/IP model is basically an idealogically or model that defines the process of how data is transmitted through the internet allowing for connectivity of nodes. TCP/IP is made up of a 5 layer tools set such as:

  1. Application layer:
    This has to do with the format in which the user would access the data, it could either be HTTP/HTTPS, SMTP and FTP.
  2. Transport layer:
    This has to do with the manner the data was transported to the user, it could be either TCP which has to do with establishing a steady connection and error free delivery or UDP which has to do with transporting the data whether or not the user get hold of everything contained in the data. in other words, UDP is connectionless.
  3. Network layer:
    This has to do with routing data packets to there destination. Data is sent using logical addresses known as IP. The data is encapsulated in an IP header which states the source where its coming from and the destination it is headed to. IP and Routers are the majors tools used in this layer.
  4. Data-link layer:
    The Data-link layer is more concerned with handling issues within the local area network. This has to do with establishing and maintaining data transfer from the physical layer to ensure error free transmission. The MAC address comes into play as a key factor in transmitting data through this layer as the data intransition is encapsulated in a MAC address header and trailer when sending making the data a frame or the MAC address is deencapsulated on receiving the frame. The MAC address is relied on to mapping out the host computer from where the data was sent and MAC address of the recipient, as the port of a switch are mapped to indicate the MAC address of the device connected to it. Ethernet, wireless access points and switches are the major tools used in the layer.
  5. Physical layer:
    This is the primary stage. It is the physical and electrical medium used to transmit data such as cables and NIC.
